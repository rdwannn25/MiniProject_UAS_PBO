# Smart Parking System

## Deskripsi
Smart Parking System adalah aplikasi berbasis Java yang digunakan untuk mengelola data kendaraan parkir, menghitung biaya parkir, dan menampilkan total pendapatan parkir.

## Fitur
- Menambahkan kendaraan.
- Menampilkan daftar kendaraan.
- Menghitung total pendapatan.
- Menangani error ketika kapasitas parkir penuh.

## Cara Menjalankan Program
1. Pastikan JDK 17 telah terinstal.
2. Compile seluruh file Java:
   javac *.java
3. Jalankan program:
   java Main
